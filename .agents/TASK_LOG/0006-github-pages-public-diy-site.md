## Goal

Add a credibility-first static GitHub Pages site for public sharing of curated
DIY materials from this ESP32 workspace.

## Scope

Included: static Pages source under `site/github-pages/`, stdlib build script,
generated artifact allowlist, public file manifest, admin HMI demo copy,
GitHub Actions Pages workflow, source-index entries, deployment-model docs,
scaffold verifier updates, validation, and handoff.

Excluded: live GitHub deployment, repository settings changes, firmware source,
framework project files, vendor PDFs, raw photo publishing, generated
screenshots, relay switching, XBee writes, firmware flashing, and mains/load
wiring.

## Decisions

- Pages deploys from the generated `build/github-pages` artifact, not from the
  repository root or the working `docs/` tree.
- The public artifact is allowlist-only and generated by
  `scripts/build_github_pages.py`.
- The site remains dependency-free static HTML/CSS/JS/JSON with no CDNs,
  frontend framework, vendor PDFs, bulky assets, or raw upload archive
  publishing.
- All public links are relative because no owner/repo URL is hard-coded.
