# Agent Coordination

## Required sequence

1. Read `AGENTS.md`.
2. Identify the owner role in `.agents/OWNERSHIP.md`.
3. Open or update a task record in `.agents/TASK_LOG/`.
4. Check source coverage before editing factual docs.
5. Leave validation evidence and handoff notes.

## Enforcement

No feature or factual document should be considered accepted unless it has:

- a task record,
- a source-index reference where factual claims are made,
- explicit unknowns,
- validation results,
- and an owner for the next action.
