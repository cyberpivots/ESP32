# Handoff And Review

## Handoff contents

Each handoff must include:

- source role and target role,
- exact files or areas touched,
- verified sources used,
- decisions made,
- validation performed,
- unresolved gaps.

## Review focus

Reviewers should prioritize:

- unsupported hardware claims,
- hidden framework coupling,
- missing safety notes,
- unclear ownership,
- tests or validation gaps.
